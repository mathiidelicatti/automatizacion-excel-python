# Automatización de reportes en Excel con Python

Este proyecto automatiza la lectura de múltiples archivos Excel, unifica la información y genera reportes automáticamente.

## Funcionalidades

- Lectura de múltiples archivos Excel
- Consolidación de datos
- Cálculo de IVA y precio final
- Generación de reporte por producto

## Tecnologías

- Python
- pandas
- os
